# Embrace Learning – Website Development Prompt

Create a modern, lightweight, fast-loading educational website for a coaching institute named "Embrace Learning."

The website must be:

- Fully responsive (mobile-first design)
- Lightweight and optimized for speed
- SEO-friendly
- Easy to host on BigRock or any shared hosting platform
- Built using HTML5, CSS3, and minimal JavaScript (preferred for speed)
- Avoid heavy frameworks unless necessary
- No bulky animations or external heavy libraries
- Clean and professional UI
- Optimized images (use placeholders if needed)
- Simple deployment (static files that can be directly uploaded)

## 🎯 Target Audience

**Grade 6th – 10th**
- All Subjects
- Concept clarity focus
- Foundation building

**Grade 11th – 12th**
- Accounts
- Economics
- English
- Physics
- Chemistry
- Mathematics

**Skill Development Programs**
- Communication Skills
- Emotional Intelligence
- Financial Literacy
- Market Readiness Skills
- Career Readiness

Create the following sections/pages:

## 🎨 Design Requirements

- Color palette:
  - Primary: Deep Blue (#1E3A8A)
  - Secondary: Soft Orange (#F59E0B)
  - Background: White / Light Gray
- Clean typography (Google Fonts – Poppins or Open Sans)
- Proper spacing and padding
- Soft shadows, subtle hover effects
- Simple navigation bar
- Sticky header
- Footer with quick links

## ⚡ Performance Requirements

- Use minimal external libraries
- Optimize CSS
- Combine CSS into single file
- Minified JS
- Lazy loading for images
- Proper meta tags
- Favicon included
- SEO meta description and keywords

## 📱 Technical Requirements

- Fully responsive
- Use semantic HTML
- Use CSS Flexbox or Grid
- No heavy animations
- Code should be structured:
  - index.html
  - about.html
  - courses.html
  - contact.html
  - css/style.css
  - js/script.js
